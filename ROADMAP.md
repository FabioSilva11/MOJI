# Roadmap do MOJI Pet

## MVP 1 — Pet vivo na tela
- [x] Tela principal
- [x] Personagem animado por código
- [x] Humor e expressões
- [x] Necessidades básicas
- [x] Botões de interação
- [x] Loop de vida automático

## MVP 2 — Mais parecido com um pet real
- [ ] Persistência local com SharedPreferences ou Hive
- [ ] Sons por emoção
- [ ] Rotina por horário
- [ ] Notificações locais
- [ ] Reações ao acelerômetro
- [ ] Sistema de conquistas

## MVP 3 — Personalidade
- [ ] Nome do dono
- [ ] Personalidade escolhida: fofo, engraçado, tímido, bagunceiro
- [ ] Diálogos contextuais
- [ ] Memória de interações
- [ ] Conversa com IA

## MVP 4 — Monetização
- [ ] Acessórios
- [ ] Temas de quarto
- [ ] Skins de olhos
- [ ] Pacotes de sons
- [ ] MOJI Pro com IA e memória avançada
